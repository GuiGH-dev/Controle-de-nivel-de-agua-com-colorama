# Meus Sites Favoritos

Este projeto foi desenvolvido como atividade prática de HTML, com o objetivo de criar uma página web contendo alguns dos meus sites favoritos.

## Sites escolhidos

- YouTube
- Canva
- Spotify
- GitHub
- PCI Concursos

## O que a página contém

A página apresenta:

- Nome de cada site;
- Imagem representativa de cada site;
- Fonte da imagem;
- Link de acesso para cada site.
